User Guide

System Overview

The Soil Monitoring System measures soil moisture and temperature and controls a sprinkler pump based on thresholds.

Dashboard Features

  • Moisture Gauge
  • Temperature Bar
  • System Status
  • Server Status
  • Sprinkler Status
  • Runtime
  • Up-Threshold
  • Lo-Threshold
  • Client ID

Controls

  • Runtime Input
  • Up-Threshold Input
  • Lo-Threshold Input
  • Sprinkler Button

Automatic Watering Logic

  • Moisture ≤ Lo-Threshold → Pump ON
  • Moisture ≥ Up-Threshold → Pump OFF
  • Runtime limit reached → Pump OFF

Device Button

  • 1 click: Toggle pump
  • Long press: Manual override
  • 5 clicks: Reset Wi-Fi
